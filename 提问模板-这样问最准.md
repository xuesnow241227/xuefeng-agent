# 这样问，Agent 回答最准

> 把下面模板里的【括号内容】换成你自己的，直接复制发过去。

---

## 模板一：高考生选专业（最常用）

```
我是【省份】考生，选科【物理/历史/物化生/史政地】，
今年考了【分数】分，全省排名大概【位次】名。
家里是【普通工薪/做生意的/爸妈在XX行业】，【有/没有】行业资源。
想去【城市或省份，如武汉/省内/南方都行】，
最讨厌学【数学/背书/物理/都还行】，
毕业想【找个好工作赚钱多/考公务员/继续考研/稳定就行】。
帮我推荐专业和学校。
```

**示例**：
```
我是湖北考生，选科物理化学地理，今年考了580分，全省排名大概28000名。
家里是普通工薪，爸妈一年收入十来万，没有行业资源。
想去武汉或者省内，最讨厌背书，物理数学还行。
毕业想找个好工作，赚钱优先。帮我推荐专业和学校。
```

## 模板二：不知道学什么（帮你理方向）

```
我是【省份】的，今年高考，【物理/历史】类考了【分数】分。
我现在完全不知道学什么专业好，也不确定想去哪。
家庭情况是【简单描述】。
你先问我几个问题，帮我理一下方向。
```

---

## 模板三：有具体目标，让Agent评估

```
我想学【专业名称】，想去【学校或城市】。
我的情况是【省份】【物理/历史】类，考了【分数】分，排名【位次】名。
家里情况是【简单描述】。
你帮我看看这个方向合不合适？有什么坑？有没有更好的选择？
```

## 模板四：已经有几个选项，让Agent对比

```
我在纠结这几个选择：
1. 【学校A】的【专业A】
2. 【学校B】的【专业B】
3. 【学校C】的【专业C】

我是【省份】【物理/历史】类，【分数】分，排名【位次】。
家里【简单描述家庭情况】。
你帮我对比一下，哪个更适合我？为什么？
```

## 模板五：专科/低分段

```
我只考了【分数】分，只能上专科了。
家在【省份】，不想离家太远，【有没有】特别讨厌的科目。
家里【简单描述家庭情况】。
就是想毕业能找个稳定的工作。你帮我看看学什么专业好，报什么学校？
```

## 模板六：考研方向

```
我本科在【学校】读的【专业】，成绩【简单描述】。
想考研，目的是【换个好找工作的方向/想要个名校文凭/真心想做科研】。
你帮我分析一下该冲什么学校什么专业？
```

## 模板七：高一选科

```
我现在高一，马上要选科了。
以后想往【医学/计算机/金融/还没想好】方向走。
各科成绩是【简单描述强弱项】。
你帮我看看选科怎么选最合理？
```

## 小技巧

1. **信息越全越准**：省份、分数、位次、家庭情况、讨厌什么、想要什么——六样说齐了，Agent 直接给方案，不用反复追问。

2. **位次比分数重要**：如果有全省排名（位次），一定带上。分数每年波动，位次才是硬通货。

3. **讨厌什么和喜欢什么一样重要**：说清楚不能接受什么（比如"绝对不学医""看到数学就头疼"），Agent 帮你避开。

4. **家里干什么的别不好意思说**：爸妈在电力、铁路、医院、学校、做生意的——这决定了很多隐藏的好路。

5. **多轮聊效果更好**：Agent 会追问你一些问题，认真回答。聊得越深，建议越准。
