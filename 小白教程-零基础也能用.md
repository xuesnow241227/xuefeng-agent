# 雪峰Agent 使用教程 —— 给完全不懂电脑的人

> 不需要懂编程，不需要会命令行。跟着步骤走，10分钟搞定。
> 全程只需要：**复制、粘贴、双击**。遇到问题截图发我。

---

## 第一步：安装 Python（已有可跳过）

### 怎么检查有没有 Python？

按键盘 `Win + R`，输入 `cmd`，回车。黑色窗口里输入：

```
python --version
```

如果出现 `Python 3.10.x` 或 `Python 3.11.x` → **已有 Python，跳到第二步**。

如果出现"不是内部命令" → 往下看。

### 安装步骤

1. 打开浏览器，输入网址：**python.org**
2. 点黄色大按钮 **Download Python**
3. 下载完双击打开
4. ⚠️ **最重要的一步**：勾选底部的 **"Add Python to PATH"**（一定要勾！）
5. 点 **Install Now**
6. 等着，1-2 分钟装完
7. 再按 `Win+R`，输入 `cmd`，输入 `python --version`，显示版本号就对了

---

## 第二步：下载雪峰Agent

1. 打开 https://github.com/ziqihe10-droid/xuefeng-agent
2. 点绿色的 **Code** 按钮
3. 点 **Download ZIP**
4. 下载完，右键 ZIP 文件 → **解压到当前文件夹**
5. 你会看到一个 `xuefeng-agent` 文件夹，把它放桌面

---

## 第三步：获取 API Key（AI 模型的钥匙）

AI 需要联网调用大模型。推荐用 DeepSeek，便宜又好用。

1. 打开 https://platform.deepseek.com
2. 点右上角注册（用手机号就行）
3. 注册完登录，点左侧 **API Keys**
4. 点 **创建 API Key**
5. 复制那串 `sk-` 开头的钥匙（⚠️ 只显示一次，赶紧复制保存）

> 费用：DeepSeek 很便宜，聊几百次才几毛钱。新用户一般都送免费额度。

---

## 第四步：配置

1. 打开 `xuefeng-agent` 文件夹
2. 找到 `.env.example` 文件，**右键 → 重命名**，把 `.example` 删掉，变成 `.env`
3. 右键 `.env` → **打开方式 → 记事本**
4. 把里面的内容改成这样：

```
LLM_API_KEY=sk-你的钥匙粘贴在这里
LLM_BASE_URL=https://api.deepseek.com
LLM_MODEL=deepseek-chat
```

5. 保存，关闭

> ⚠️ 第三行 `sk-` 后面粘贴你刚才复制的钥匙，别留空格。

---

## 第五步：安装依赖（只需做一次）

1. 按 `Win+R`，输入 `cmd`，回车
2. 黑色窗口里输入这一行，回车：

```
pip install openai pywin32
```

3. 等它跑完（1-2分钟），出现 `Successfully installed` 就对了

> ⚠️ 这步只做一次。以后再启动不用重复装。

---

## 第六步：启动

1. 双击文件夹里的 **启动.bat**
2. 会弹出一个黑色窗口
3. 看到 `API 连接成功` 就说明一切正常
4. 直接打字问问题

---

## 怎么问？

直接像跟朋友聊天一样，告诉它你的情况：

```
河南文科520分，位次3万多，家里普通，想学个好就业的专业
```

或者：

```
我是学物理的，考了580，想去武汉，有啥推荐
```

它会先问你几个问题（家里干什么的、想去哪、讨厌什么），问清楚了再给你建议。

## 常见问题

**Q：点启动闪一下就没了？**
A：没装 Python 或者 .env 没配好。回头检查第三步和第四步。

**Q：提示 API 连接失败？**
A：检查两点：
1. API Key 有没有多复制空格？Key 是纯 `sk-` 开头的一串，前后别有空格。
2. Key 还有没有余额？去 DeepSeek 平台检查。

**Q：提示 `ModuleNotFoundError: No module named 'openai'`？**
A：没装依赖。按 `Win+R`，输入 `cmd`，执行 `pip install openai pywin32`

**Q：回答不靠谱怎么办？**
A：AI 是辅助工具，不是权威。录取数据每年变，最终决定请查学校官网和考试院官网。

**Q：想换个 AI 怎么办？**
A：打开 `.env`，改 `LLM_PROVIDER=qwen`（通义千问）或者 `LLM_PROVIDER=glm`（智谱）。Key 也要换成对应的。

**Q：Mac 电脑能用吗？**
A：能。打开终端，cd 到文件夹，`python agent.py` 就行。
